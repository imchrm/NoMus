Пользователь -> /start
    -> presentation.bot.handlers.common.cmd_start
        -> Отображает кнопки "Регистрация" и "Сделать заказ"

Пользователь -> нажимает "Сделать заказ"
    -> presentation.bot.handlers.ordering.start_ordering
        -> application.services.auth_service.is_user_registered (проверка регистрации)
        -> application.services.order_service.get_tariffs (получение тарифов)
        -> Отображает список тарифов

Пользователь -> выбирает тариф
    -> presentation.bot.handlers.ordering.process_tariff
        -> Отображает кнопку "Оплатить {сумма}"

Пользователь -> нажимает "Оплатить"
    -> presentation.bot.handlers.ordering.process_payment
        -> application.services.order_service.create_order (создание заказа)
        -> Отправляет сообщение "Заказ успешно оплачен и создан!"
